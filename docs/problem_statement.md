# Problem statement (course project)

Given an artificial dataset with **100 features**, build a **binary classifier** and output a vector of
**probabilities of belonging to class 1** for all observations in the test set.

- Train set: 1500 rows  
- Test set: 500 rows  
- Target labels for train set: `Class` (values: 1 or 2)

The main evaluation metric used in the course was **balanced accuracy**.
